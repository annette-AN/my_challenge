css

attr 이용

  • enabled/disabled 상태 이용가능한 요소
    button, input, seclect, option, textarea

    button {cursor: pointer;}
    button:disabled {opacity:0.8; cursor: initial;}

    (:enabled 는 defult 상태로 따로 선언해서 사용하지는 않지만 선언은 가능
    :disabled 는 기본적인 비활성 디자인이 적용되어 있으며, 비활성 디자인 추가가 필요할 때 사용)

  • checked 상태 이용가능한 요소 (it also has enabled, disabled)
    input[type="radio"], input[type="checkbox"], option

    input[type="radio"]:checked {outline: 1px dashed red;}

  • [추가필요] :visited [접근을 했었는지 표시], :focus, :hover, :active [mousedown ~ mouseup], :empty [text가 없는 것을 인지], :not(~) [~가 아닐 때 전부]

class 이용

  • column layout
    css table or flex 속성 사용
    class rule

    .col {width: auto;}
    .col_full {width: 100%;}
    .col_half {width: 50%;}
    .col_third {width: 33.3%;}
    .col_querter {width: 25%;}

    .col > .box {width: 100%;}

  • row layout
    class rule

    .row {width: 100%;}

    .row > .inner_section {width:100%; max-width: 1200px;}
    .row > .outer_section {width: 100%;}

  • [추가필요].box visual 적인거
  • 의미를 가진 design
    class rule

    .(selector)_danger (위험)

    .(selector)_darken (반전되는 design)
